# PyBattle (xBattle resurrection)

inspired by https://inf.ug.edu.pl/~piotao/old/xbattle/ (in Polish)

## Requirements and launching instructions

##### Python:
we recommend 3.7.4 or 3.6.9, because we are using them.

##### Python packages:
* Pyglet
* numpy

##### Supported OS:
* Windows
* Linux
* MacOS

##### Run the game:
```
python window.py
```
